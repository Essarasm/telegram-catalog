Per-client current debt from latest 1C 'Дебиторская задолженность' upload. Real clients only (pseudo-accounts excluded). Sorted by combined USD-equivalent debt. Agent — manually assign the responsible agent from the dropdown; defaults to "—" until you (or another admin) pick one. Select "—" to clear. Days since = today − last_transaction_date; large values indicate stuck debt. Qayta qo'ng'iroq — pick a date when the client asked to be called back; the row tints amber when that date arrives. The tint fades automatically once a new order or payment lands dated after the callback was set (the client kept their promise). History is preserved (every reschedule appends).
